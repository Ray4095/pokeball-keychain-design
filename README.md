# pokeball-keychain-design

This repository contains a 3D keychain model designed using Onshape.

## Project Description

The design is a Pokeball-inspired keychain with a circular base, a keychain hole, a raised middle band, and a raised center button.

The model was created with estimated dimensions and exported as an STL file.

[View the design on Onshape](https://cad.onshape.com/documents/00d5a3d330022b8a0c4f8694/w/8336315fdf836acd87703994/e/dc506a9a60eca2bd09f9d20c?renderMode=0&uiState=6a5ada3dfb9b884c4aa4d625)

## Tools Used

- Onshape

## Design Features

- Circular keychain base
- Keychain hole near the top
- Raised middle band
- Raised center button
- Colored design preview
- STL file exported for 3D model use

## Estimated Dimensions

- Outer diameter: 60 mm
- Base thickness: 4 mm
- Keychain hole diameter: 8 mm
- Keychain hole distance from center: 22 mm
- Middle band height: approximately 6.8 mm
- Center button diameter: 16 mm
- Raised features height: 2 mm
